Process and Visualize Data with R

This course is designed to help air quality analysts process and visualize data using R. In this course you will learn how to download and import AQS data inside of R. You will also learn how to process data using the dplyr and tidyr packages. The course will cover how to quickly subset, filter, summarize, and reshape data. You will learn how to create maps in R, as well as visualize air quality data using the openair package. The training also shows you how to build custom web applications with interactive plots and tables using the shiny package.
